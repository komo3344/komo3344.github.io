> npm 최신버전으로 업데이트하기  
버전별로 업데이트 방법이 다르다. 가장 최신에 릴리즈된 것은 next지만 안정성은 latest가 좋다.  
- latest버전: npm install -g npm@latest  
- next버전: npm install -g npm@next
